# Mental State Clarifier

This project explores using large language models to help individuals articulate their mental state more clearly, particularly in early-stage mental health contexts. It aims to bridge the communication gap between patients and providers by structuring and clarifying emotional descriptions.

## Key Features
*   Processes user-inputted text about emotional state.
*   Uses an LLM to generate structured summaries and clarifying questions.
*   Outputs a refined, coherent description of mental state.
*   Designed as a lightweight, prototype-level tool for exploration.

## Tech Stack
*   Python
*   OpenAI API (or compatible LLM)
*   Streamlit (for prototype interface)

## Getting Started
1.  Clone the repo: `git clone https://github.com/zoreanuj/mental-state-clarifier.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Set your API key: `export OPENAI_API_KEY='your-key'`
4.  Run the app: `streamlit run app.py`
